One Skillet Mediterranean Chicken

Description

This is a delicious and healthy mediterranean diet meal. The original recipe can be found on themediterraneandish.com.

I chose to add this meal because it is so healthy and refreshing, but at the same time, extremely simple and can be made in a single pan. Try chicken breasts for a healthier lean cut or chicken thighs for a more flavorful fattier cut. Diced tomatoes can be fresh or canned, but I prefer fresh.
